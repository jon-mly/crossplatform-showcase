# Cross-Platform Showcase

A comparison of cross-platform mobile development frameworks by implementing the same simple CRUD application across different technologies.

## Purpose

Showcase best practices for mobile app development:
- Clean Architecture
- SOLID principles
- Testing strategies
- Code organization and structure

## The App

A basic employee management app with:
- List of employees
- Create, edit, and delete employees
- API integration

## Implementations

| Framework | Status |
|-----------|--------|
| Flutter | Done |
| Compose Multiplatform (KMP) | Planned |
| React Native | Planned |
| iOS Native (Swift) | Optional |
| Android Native (Kotlin) | Optional |
